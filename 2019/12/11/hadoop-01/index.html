HDFS的API

2019-12-11
hadoop

使用HDFS的API操作集群上的文件

简单来说，就3步

  1. 创建一个配置信息对象，对象用于获取HDFS文件系统的对象（单例模式）
  2. 调用 HDFS的对象 的对应的上传下载等操作，需要什么参数就创建什么参数
  3. 关闭资源

下面看一个示例

假设我们获取的 HDFS的文件系统的对象名为fs，那我们来看下它的一些API操作

上传 fs.copyFromLocalFile(src, dst); //src 和 dst都是Path类型，分别表示源路径和目的路径

下载 fs.copyToLocalFile(false, src, dst, true); /
// boolean delSrc 指是否将原文件删除
// Path src 指要下载的文件路径
// Path dst 指将文件下载到的路径
// boolean useRawLocalFileSystem 是否开启文件效验

创建目录 fs.mkdir(Path p1)

删除目录 fs.delete(Path p1,boolean b) //如果是空文件夹，参数2必须给值true

文件名更改 fs.rename(Path p1,Path p2)
